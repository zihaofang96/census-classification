# Census data classification and segmentation

This project builds a classification model and a segmentation model based on a census dataset from the 1994 and 1995 Current Population Surveys by the U.S. Census Bureau. The classication model predicts the income level of individuals, using information about their demographics, occupation, education, etc. The segmentation model splits the data points into clusters, each of which have some clear charateristics that one can use for potential business strategies.

## Description of the files

The data is stored in "census-bureau.data", with the header saved in "census-bureau.columns".

The classification model is contained in "classification.ipynb" and the segmentation model is built and analyzed in "segmentation.ipynb". Code could be compiled sequentially in Jupyter notebook.

A summary of the project can be found in "Project Report.pdf".

The required packages could be installed into an environment using "environment.yml".
